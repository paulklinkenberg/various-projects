AddList: Sdds a mailing list to the specified domain
o Return Type: Generic Result
o Parameters:
• AuthUserName: User name used to perform this function. Requires system
administrator or domain administrator privileges.
• AuthPassword: Password for the username
• DomainName: The domain name for which to add the list
• ListName: Name of the list to add
• Moderator: Moderator of the list
• Description: Description of the list
