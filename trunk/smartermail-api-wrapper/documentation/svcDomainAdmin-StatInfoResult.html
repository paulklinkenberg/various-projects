StatInfoResult: Returned by GetDomainStatistics
o Result: True or false depending on the success of the Web service request
o ResultCode: Specific failure code if the result is false
o Message: Error message detailing why the result is false
o MessagesReceived: Total number of messages received in that time period
o MessagesSent: Total number of messages sent in that time period
o BytesSent: Total number of bytes sent in that time period
o BytesReceived: Total number of bytes received in that time period
o BytesSize: Total disk space used by the domain
