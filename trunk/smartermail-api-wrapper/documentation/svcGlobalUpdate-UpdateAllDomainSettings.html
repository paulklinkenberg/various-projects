UpdateAllDomainSettings: Sets the supplied fields and values for all domains on the server.
o Return Type: GenericResult
o Parameters:
• AuthUserName: User name used to perform this function. Requires system
administrator privileges.
• AuthPassword: Password for the username
• Fields: An array of DomainSettingFieldStruct structs. This struct has a property
for the key and the value of the setting to change. For a list of valid keys, call the
ListGlobalUpdateFields function.
