The elements with a red border are <pre> tags.
The javascript finds all pre tags, takes the contents/innerHtml, and sends it to the server (doColorCoding.cfm) by using ajax.
The server then returns an xml file with the color-coded text.
With the xml data, the javascript changes the contents of the pre-tags with the xml data.

For more info, see http://www.coldfusiondeveloper.nl/post.cfm/automatic-coldfusion-code-coloring-on
