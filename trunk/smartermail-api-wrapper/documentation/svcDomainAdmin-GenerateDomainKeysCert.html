GenerateDomainKeysCert : Generates a domainkeys certificate for the domain
o Return Type: DomainKeyResult
o Parameters:
• AuthUserName: User name used to perform this function. Requires system
administrator privileges
• AuthPassword: Password for the username
• DomainName: Domain for which the certificate should be created
• KeySize: Size of the certificate key (512/768/1024/1536/2048 bit)
• Selector: DomainKeys selector to use
