UpdateUserForwardingInfo: updates the specified user's forwarding settings
o Return Type: GenericResult
o Parameters:
• AuthUserName: User name used to perform this function. Requires system
administrator or domain administrator privileges
• AuthPassword: Password for the username
• EmailAddress: Account to Update
• DeleteOnForward: Should the account delete emails it forward
• ForwardingAddress: Address to which forwards should be sent
