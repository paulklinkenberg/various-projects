AddUserGroup: Creates a new user group adds it to the specified domain
o Return Type: GenericResult
o Parameters:
• AuthUserName: User name used to perform this function. Requires system
administrator or domain Aadministrator privileges
• AuthPassword: Password for the username
• DomainName: Domain to which the user group should be added
• UserGroupName: Name of the usergroup
• UserNames: Array of usernames that should belong to the group
