SetDomainForwardingBlock: Updates the target address and domains in a domain forwarding
block
o Return Type: GenericResult
o Parameters:
• AuthUserName: User name used to perform this function. Requires system
administrator privileges
• AuthPassword: Password for the username
• blockID: The ID of the forwarding block to update
• targetAddress: The new target address for the domain forwarding block
• domains: An array of domain names that will be forwarded to the specified
target address
