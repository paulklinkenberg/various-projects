AddRSSFeed: Adds the specified RSS feed to the account
o Return Type: GenericResult
o Parameters:
• AuthUserName: User name used to perform this function. Requires system
administrator or domain administrator privileges to change an account other
than itself
• AuthPassword: Password for the username
• EmailAddress: Address for which to add the feed
• rssUrl: Url of the RSS feed
• rssName: Name of the RSS feed
• rssFolder: Folder to place the RSS feed in
