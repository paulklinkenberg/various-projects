UpdateGateway: Updates the settings for a specific gateway
o Return Type: GenericResult
o Parameters:
• AuthUserName: User name used to perform this function. Requires system
administrator privileges
• AuthPassword: Password for the username
• Address: Address of the gateway server
• Enabled: Should SmarterMail use this gateway
• smURL: Url to the SmarterMail Web interface
• smUserName: Admin username to the SmarterMail gateway
• smPassword: Password to the SmarterMail gateway
