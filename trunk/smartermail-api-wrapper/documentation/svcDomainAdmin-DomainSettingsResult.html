DomainSettingsResult: Returned by GetDomainDefaults and GetDomainSettings
o Result: True or false depending on the success of the Web service request
o ResultCode: Specific failure code if the result is false
o Message: Error message detailing why the result is false
o ServerIP: The IP the domain is listening on
o Path: The storage location of the domain’s data
o ImapPort: The port this domain should accept IMAP connections on
o PopPort: The port this domain should accept POP connections on
o SmtpPort: The port this domain should accept SMTP connections on
o MaxAliases: The maximum number of aliases this domain is allowed to have (0 =
unlimited)
o MaxDomainSizeInMB: The maximum size this domain is allowed to occupy (0 =
unlimited)
o MaxDomainUsers: The maximum number of users this domain is allowed to have (0 =
unlimited)
o MaxMailboxSizeInMB: This parameter is no longer used
o MaxMessageSize: The maximum size of a message users on this domain are allowed to
send. (0 = unlimited)
o MaxRecipients: The maximum number of recipients that are allowed in a single email (0
= unlimited)
o MaxDomainAliases: The maximum number of domain aliases this domain is allowed to
have (0 = unlimited)
o MaxLists: The maximum number of mailing lists this domain is allowed to have (0 =
unlimited)
o ShowDomainAliasMenu: If this is set to false, domain admins will not be able to
manage domain aliases
o ShowContentFilteringMenu: If this is set to false, users will not be able to use content
filtering
o ShowSpamMenu: If this is set to false, users will not be able to manage their own spam
settings
o ShowStatsMenu: This parameter is no longer used
o RequireSmtpAuthentication: If this is set to true, the domain will require SMTP
Authentication to relay emails
o ShowListMenu: If this is set to false, domain admins will not be able to manage mailing
lists
o ListCommandAddress: This is the command address for mailing lists
