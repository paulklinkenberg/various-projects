AddDomainAliasWithoutMxCheck: Creates a new domain alias and adds it to the specified
domain without doing an MX lookup on the new alias.
o Return Type: GenericResult
o Parameters:
• AuthUserName: User name used to perform this function. Requires system
administrator privileges.
• AuthPassword: Password for the username
• DomainName: Domain name for which to create an alias
• DomainAliasName: The name of the alias
