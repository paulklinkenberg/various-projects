UpdateDomainNameAndPath: changes the specified domain's name and/or path.
o Return Type: GenericResult
o Parameters:
• AuthUserName: User name used to perform this function. Requires system
administrator privileges
• AuthPassword: Password for the username
• OldDomainName: The current name of the domain
• NewDomainName: The new name of the domain
• NewDomainPath: The new path for the domain
