SetRequestedListSettings: sets the requested settings for the specified list.
o Return Type: GenericResult
o Parameters:
• AuthUserName: User name used to perform this function. Requires system
administrator or domain administrator privileges.
• AuthPassword: Password for the username
• DomainName: The domain name to which the list belongs
• ListName: Name of the list to update
• newSttings: An array of key/value pairs in the form of key=value.
