GatewayResult: Returned by GetGateway
o Result: True or false depending on the success of the Web service request
o ResultCode: Specific failure code if the result is false
o Message: Error message detailing why the result is false
o Guid: Unique ID of the gateway
o gatewayAddress: Address of the gateway
o enabled: Is this gateway enabled
o isSmarterMailServer: Is this a SmarterMail gateway
o smUrl: Url to the SmarterMail Web interface of the gateway
o smUserName: Username for the SmarterMail Gateway
o smPassword: Password for the SmarterMail Gateway
