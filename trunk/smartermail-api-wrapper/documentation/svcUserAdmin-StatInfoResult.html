StatInfoResult: Returned by GetUserStats
o Result: True or false depending on the success of the Web service request
o ResultCode: Specific failure code if the result is false
o Message: Error message detailing why the result is false
o MessagesReceived: Total messages received in time frame
o MessagesSent: Total messages sent in time frame
o BytesReceived: Total bytes received in time frame
o BytesSent: Total bytes sent in time frame
o BytesSize: Disk space usage by user
